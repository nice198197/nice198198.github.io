本节主要介绍common相关api。

common

math

  • b2Vec2：二维向量
  • b2Mat22：一个由两个b2Vec2组成的2*2方阵，可直接由两个b2Vec2（col1、col2）构造或者由一个角度值构造
  • b2Math：基本数学运算，如向量和矩阵的计算、产生随机数等

bsSettings

  • bsSettings：基础系统配置，如pi的值、每秒的时间、每米的大小、睡眠时间等等
